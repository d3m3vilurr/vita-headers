vitasdk - Documentation

Abstract

This documentation contains User and Kernel functions exported by Vita modules.
It is aimed at people who want to write applications and plugins for the Vita with the vitasdk and want to get an overview about the functions they can use.

License: The header files of the vitasdk are released under the terms of the https://github.com/vitasdk/vita-headers/blob/master/LICENSE.md "MIT license", here is a non-legally binding summary of what you can do.
